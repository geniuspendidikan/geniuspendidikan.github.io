2. jelaskan pengertian dari istilah web dan homepage!


Pengertian dari istilah web adalah suatu layanan dalam internet yang berisikan beragam informasi yang dapat diakses oleh pengguna komputer yang terhubung. Dalam hal ini, sekumpulan web dapat disebut dengan website. Berbeda dengan homepage, homepage merupakan interface utama dari suatu website pada internet dan umumnya homepage dapat menjadi daya tarik utama pengunjung melalui desain tampilannya. Contoh homepage seperti pada www.brainly.co.id.

Pembahasan

Web dan homepage merupakan bagian dari layanan internet yang berhubungan dengan informasi. Selain homepage dan web, dikenal pula istilah webpage dan website. Keseluruhan hal tersebut dapat diakses melalui bantuan peramban atau browser. Perbedaan homepage dengan webpage sendiri jika dianalogikan adalah cover dan isinya. Homepage lebih mengrah halaman depan dan webpage merupakan bagian-bagiannya.

Contoh : dalam Brainly, homepagenya adalah www.brainly.co.id yang berisi tampilan utama dan menu pilihan mapel, notifikasi, profil dan leaderboard sedangkan webpagenya dapat berupa www.brainly.co.id/profil/username-id  maupun www.brainly.co.id/tugas/idtugas  nantinya webpage ini akan digabungkan dan terkait sehingga disebut dengan website

Sedangkan, sekumpulan web dapat dikatakan sebagai website yang menggunakan www sebagai awalan dan diakhiri dengan domain seperti .com .org .co.id

26. jelaskan perbedaan antara URL, WEBSITE, HOMEPAGE DAN WEBPAGE


Kelas : X SMA
Pelajaran : TI
Kategori : Cara Menggunakan Internet
Kata kunci : URL, website, homepage, web page

Penjelasan :

URL

URL (Uniform Resource Locator) adalah cara penamaan alamat file di Internet. URL merupakan serangkaian karakter (dapat berupa huruf, angka, ataupun simbol) yang sesuai dengan format standar yang sudah ditentukan, URL digunakan untuk menunjukkan alamat/ address suatu sumber yang ada.

WEBSITE

Website adalah kumpulan dari berbagai macam halaman situs, yang terangkum didalam sebuah domain atau juga subdomain, yang lebih tempatnya berada di dalam WWW (World Wide Web) yang tentunya terdapat di dalam Internet.

Macam-macam domain website
- .co.id : Biasanya digunakan untuk badan usaha yang memiliki badan hukum sah.
- .go.id : Khusus digunakan untuk Lembaga Pemerintahan RI.
- .ac.id : Dipakai untuk Lembaga Pendidikan.
- .or.id: Dipakai untuk segala macam organisasi yang tidak termasuk kedalam kategori ”co.id”,”go.id”,”mil.id”, “ac.id” dan sebagainya.
- .war.net.id : Dipaki untuk industri warung internet (warnet) yang ada di Indonesia
- .sch.id: Dipakai khusus untuk Lembaga Pendidikan SD, SMP dan SMU atau SMK
- .web.id: Biasanya digunakan untuk organisasi, badan usaha, ataupun perseorangan yang melakukan kegiatannya di WWW.

HOMEPAGE

Homepage adalah halaman muka dari seuatu situs web (web site), atau lokasi dari situs web di mana dokumen web dapat diakses. Definisi yang lainnya yaitu halaman default yang telah diset untuk sebuah browser, dimana halaman ini ditayangan di Internet melalui media World Wide Web (WWW).

Fungsi homepage (halaman utama atau awal) :
- menampilkan serta mengakses halaman awal dari suatu situs
- dapat menampilkan pada topik fitur, sebab merupakan bagian dari fitur yang terdapat pada situs tertentu tergantung dari settingan pemilik atau adminnya.
- sebagai halaman utama dari suatu situs web.

WEBPAGE

Web page adalah halaman yang ditampilkan di sebuah website di internet, yang dapat menampilkan teks, gambar, bahkan suara. Halaman ini digunakan untuk menunjukan sebuah halaman website.
Web page dapat diibaratkan sebuah halaman dalam buku.
web page biasanya berupa script yang di awali dengan HTML

Browser adalah suatu aplikasi atau program yang dijalankan pada perangkat komputer untuk melihat konten yang ada pada media World Wide Web (WWW) dengan memanfaatkan jaringan internet.
