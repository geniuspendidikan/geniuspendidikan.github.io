5. tempat parkir seluas 720 m2 hanya mampu menampung mobil dan bus sebanyak 80 kendaraan tidak memerlukan tempat 6 m2 dan bus 12 m2 biaya parkir tiap mobil Rp5.000 dan bus Rp7.500 jika tempat parkir penuh pendapatan maksimum dari biaya parkir adalah


Jika tempat parkir penuh pendapatan maksimum dari biaya parkir adalah Rp. 500.000. Soal ini terkait program linear dua variabel.

Pembahasan

Kita misalkan,

mobil = x

bus = y

maka,

x + y ≤ 80 (i)

6x + 12y ≤ 720

x + 2y ≤ 120 (ii)

Setelah itu akan digambar grafiknya dengan langkah berikut:

Pertama, jika x = 0, maka di pertidaksamaan (i) menjadi

x + y = 80

0 + y = 80

y = 80

berarti (0, 80)

kalau di pertidaksamaan (ii) menjadi

x + 2y = 120

0 + 2y = 120

2y = 120

y = 120/2 = 60,

berarti (0, 60)

Kedua, jika y = 0, maka

x + y = 80

x + 0 = 80

x = 80

berarti (80, 0)

x + 2y = 120

x + 0 = 120

x = 120

berarti (120, 0)

Dari koordinat-koordinat yang didapat, kita mendapatkan gambar grafik seperti pada lampiran.

Dari kurva itu kita ambil beberapa titik potong untuk mencari biaya parkir maksimum. Titik potong yang kita ambil adalah (0, 60), (80, 0) dan titik potong kedua kurva.

(0, 60)

5.000x + 7.500y

5.000 (0) + 7.500(60)

0 + 450.000 = Rp 450.000

(80, 0)

5.000x + 7.500y

5.000(80) + 7.500(0)

400.000 + 0 = Rp 400.000

Titik potong kedua kurva

x + y = 80

x + 2y = 120

---------------- -

-y = -40

y = 40

x + y = 80

x + 40 = 80

x = 80 - 40 = 40

(40, 40)

5.000x + 7.500y

5.000 (40) + 7.500 (40)

200.000 + 300.000 = Rp. 500.000

Dari ketiga koordinat yang kita ambil ternyata 40 mobil dan 40 bis yang menghasilkan pendapatan maksimum yaitu Rp. 500.000.

mobil = a 
bus = b

luas parkir = jumlah mobil + jumlah bus
600 = 6a + 24 b       (x1000)
155000 = 2000a + 5000b    (x3)

gunakan eliminasi 
600000 = 6000a + 24000b
465000 = 6000a + 15000b     _
----------------------------------------
135000 = 9000b
b = 15

substitusikan
6a + 24b = 600
6a + 24(15) = 600
6a = 600 - 360
6a = 240
a = 40

jumlah mobil = 40
jumlah bus = 15

mobil = x
bus = y
6x+24y=600                x=100-4y
2000x+5000y=155000

200000-8000y+5000y=155000

15. suatu tempat parkir dipenuhi tiga jenis kendaraan yaitu, sepeda motor, mobil , dan mini bus. luas parkir mini bus adalah lima kali luas parkir sepeda motor, sedangkan tiga kali luas parkir untuk mobil sama dengan luas parkir untuk mini bus dan sepeda motor. jika tempat parkir penuh dengan banyak kendaraam yang terparkir sebanyak 180, maka banyak setiap kendaraan yang parkir adalah


Kategori soal : MATEMATIKA - Sistem Persamaan Tiga Variabel
Kelas : SMA
==================================================

Permasalahan dalam soal di atas dapat diselesaikan dengan menggunakan konsep sistem persamaan tiga variabel.

Langkah I:
--------------

Langkah pertama adalah memisalkan variabel bebas yang ada dalam soal.

Misalkan:
(i) luas parkiran untuk sebuah mini bus adalah x
(iiI luas parkiran untuk sebuah sepeda motor adalah y
(iii) luas parkiran untuk sebuah mobil adalah z

Langkah II:
---------------

Langkah kedua adalah menyatakan kalimat informasi dalam soal ke dalam persamaan matematika.

1. luas parkir mini bus adalah lima kali luas parkir sepeda motor → x = 5y 

2. tiga kali luas parkir untuk mobil sama dengan luas parkir untuk mini bus dan sepeda motor → z = x + y = 5y + y = 6y

3. jika tempat parkir penuh, maka banyak kendaraam yang terparkir adalah sebanyak 180 kendaraan → x + y + z = 180
                                              5y + y + 6y = 180
                                             12y = 180
                                              y = 15

Langkah III:
---------------

Mensubtitusikan y = 15 ke poin 1 dan 2, sehingga diperoleh nilai x dan z.

x = 5y = 5(15) = 75

z = 6y = 6(15) = 90

Berdasarkan uraian di atas, dapat disimpulkan bahwa banyak mini bus, sepeda motor, dan mobil yang mungkin dalam parkiran tersebut berturut-turut adalah 75, 15, dan 90 kendaraan.

Pembahasan:

Untuk menyelesaikan soal seperti ini kita buat terlebih dahulu kalimat matematikanya, dan kita tentukan nilai untuk variabel x dan nilai untuk variabel y.

Pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.

Bentuk pertidaksamaan linier dapat ditulis:

ax + by ≤ c

ax + by ≥ c

ax + by < c

ax + by > c

Dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran

Dalam soal, kita buat pertidaksamaannya terlebih dahulu:

Bus = x

Mobil = y

x + y ≤ 58

24x + 6y ≤ 600

z = 750x + 500y

Ditanya

Hasil dari biaya parkir maksimum = ...?

Jawab:

Kita buat nol fungsi terlebih dahulu

x + y ≤ 58

Memotong sumbu x di titik y = 0

x + y = 58

x + 0 = 58

x = 58

Memotong sumbu y di titik x = 0

x + y = 58

0 + y = 58

y = 58

24x + 6y ≤ 600 kita kecilkan dengan membagi 6 pada semua ruas

4x + y ≤ 100

Memotong sumbu x apabila y = 0

4x + y = 100

4x + 0 = 100

4x = 100

x = 100/4

x = 25

Memotong sumbu y apabila x = 0

4x + y = 100

0 + y = 100

y = 100

Langkah selanjutnya kita eliminasi

4x + y = 100

x + y = 58

--------------------- -

    3x = 42

      x = 42/3

      x = 14

x + y = 58

14 + y = 58

y = 58 - 14

y = 44

kita tentukan titik uji yang memenuhi

kita dapat 3 titik uji, perhatikan gambar pada lampiran

(0, 58)(14, 44)(25, 0)

Subsitusikan titik uji pada nilai z = 750x + 500y

1) Titik uji pertama (0, 58)

   z = 750x + 500y

     = 750(0) + 500(58)

     = 0 + 29.000

     = 29.000

2) Titik uji kedua (14, 44)

   z = 750x + 500y

      = 750(14) + 500(44)

      = 10.500 + 22.000

     = 32.500

3) Titik uji ketiga (58, 0)

   z = 750x + 500y

     = 750(25) + 500(0)

     = 18.750 + 0

     = 18.750

Pendapatan maksimum tempat parkir yaitu 32.500 dengan 14 bus dan 44 mobil yang parkir.

Pembahasan

Diketahui :

Tempat parkir seluas 600 m^2

hanya mampu menampung 58 bus dan mobil

Tiap Mobil membutuhkan tempat 6 m^2 dan bus 24 m^2

hasil dari biaya parkir maksimum = ... ?

Jawab :

Misalkan bus dilambangkan dengan huruf b

dan mobil dilambangkan dengan huruf m

Pertama menggunakan rumus eliminasi

m + b = 58 | ×6 |

6m + 6b = 348

6m + 24b = 600

_______________--

-18b = -252

b = -252/-18

b = 14

Kedua menggunakan rumus subsitusi

m + b = 58

m + 14 = 58

m = 58 - 14

m = 44

44(500) + 14(750)

= 22.000 + 10.500

= 32.500

Jadi, hasil biaya parkir maksimum adalah Rp 32.500
