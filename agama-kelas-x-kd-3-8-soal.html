Jawaban Pendahuluan

Untuk menggambar garis pada bidang kartesius dibuthkan minimal dua titik. Dari dua titik tersebut lalu ditarik garis. Agar lebih mudah, titik yang dipakai adalah x = 0 dan y = 0. Semakin banyak titik yang digunakan maka grafik akan semakin akurat dan bagus.  

Pembahasan

a. 2x = 6y

saat x = 0, y = 0saat x = 3, y = 1           saat x = -3, y = -1saat x =6,  y = 2          saat x = -6,  y = -2Gambar dapat dilihat pada gambar 1

b. 3y - 4  = 4y

y = - 4Gambar dapat dilihat pada gambar 2

c. 4x + 2y = 6

saat x = 0, y = 3saat y = 0, x = 1,5saat x = 1,  y = 1           Gambar dapat dilihat pada gambar 3

c. y + 3x - 4 = 0

saat x = 0, y = 4saat y = 0, x = 1,33saat x = 1,  y = 1           Gambar dapat dilihat pada gambar 4Kesimpulan

Untuk menggambar garis pada bidang kartesius dibuthkan minimal dua titik. Titik tersebut dicari dengan memisalkan nilai x atau y pada persamaan
