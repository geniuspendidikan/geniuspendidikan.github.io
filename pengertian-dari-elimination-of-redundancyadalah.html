15. MATRIX 4x41. Solve the following system of equations using the Gauss elimination method:2x¹+x²-x³+4x⁴=19-x¹-2x²+x³+2x⁴=-32x¹+4x²+2x³+x⁴=25-x¹+x²-x³-2x⁴=-5​


Step 1: Write the augmented matrix for the system of equations

| 2 1 -1 4 | 19 |

| -1 -2 1 2 | -3 |

| 2 4 2 1 | 25 |

| -1 1 -1 -2 | -5 |

Step 2: Use the Gauss elimination method to transform the matrix into row echelon form

| 2 1 -1 4 | 19 |

| 0 -1 1/2 9/2 | 35/2 |

| 0 0 3 -11 | -17 |

| 0 0 0 -3/2 | -1/2 |

Step 3: Solve for the variables starting from the last row

-3/2x⁴ = -1/2

x⁴ = 1/3

3x³ - 11x⁴ = -17

3x³ - 11(1/3) = -17

3x³ = -20/3

x³ = -20/9

-x² + 1/2x³ + 9/2x⁴ = 35/2

-x² + 1/2(-20/9) + 9/2(1/3) = 35/2

-x² = 145/18

x² = -145/18

2x¹ + x² - x³ + 4x⁴ = 19

2x¹ + (-145/18) - (-20/9) + 4(1/3) = 19

2x¹ = 509/18

x¹ = 509/36

Step 4: Write the solution set

x¹ = 509/36

x² = -145/18

x³ = -20/9

x⁴ = 1/3

Therefore, the solution set for the given system of equations is (509/36, -145/18, -20/9, 1/3).

34. Three tables and two chairs cost Rp1.900 and two tables and four chairs cost Rp1.800. Find the cost of table and a chair. (KE ELIMINATION METHOD)​


Jawaban:

To find the cost of a table and a chair using the elimination method, you can set up a system of equations based on the given information. Let T represent the cost of one table and C represent the cost of one chair.

From the information provided, we have two equations:

1. Three tables and two chairs cost Rp1.900:

3T + 2C = 1900

2. Two tables and four chairs cost Rp1.800:

2T + 4C = 1800

Now, you can use the elimination method to solve this system of equations. Multiply the first equation by 2 to make the coefficients of T in both equations equal:

1. 2(3T + 2C) = 2(1900)

6T + 4C = 3800

Now, you have two equations with the same coefficient for T:

1. 6T + 4C = 3800

2. 2T + 4C = 1800

Now, subtract the second equation from the first equation to eliminate C:

(6T + 4C) - (2T + 4C) = 3800 - 1800

4T = 2000

Now, divide both sides by 4 to solve for T:

4T/4 = 2000/4

T = 500

Now that you know the cost of one table (T = 500), you can substitute it back into one of the original equations to find the cost of one chair. Let's use the first equation:

3T + 2C = 1900

3(500) + 2C = 1900

1500 + 2C = 1900

Subtract 1500 from both sides:

2C = 1900 - 1500

2C = 400

Now, divide both sides by 2 to solve for C:

2C/2 = 400/2

C = 200

So, the cost of one table is Rp500, and the cost of one chair is Rp200.
