def bubble_sort(array):

   for i in range(len(array) - 1):

       for j in range(len(array) - i - 1):

           if array[j] > array[j + 1]:

               array[j], array[j + 1] = array[j + 1], array[j]

   return array

array = [11, 9, 6, 10, 9, 1, 3, 5, 2]

sorted_array = bubble_sort(array)

print(sorted_array)

2. selection sort:

b. start (flowchart)

initialize i = 0

while i < len(array) - 1

   initialize min_index = i

   for j = i + 1 to len(array)

       if array[j] < array[min_index]

           set min_index = j

   swap array[i] and array[min_index]

   increment i by 1

end

print sorted array

kode program:

def selection_sort(array):

   for i in range(len(array)):

       min_index = i

       for j in range(i + 1, len(array)):

           if array[j] < array[min_index]:

               min_index = j

       array[i], array[min_index] = array[min_index], array[i]

   return array

array = [11, 9, 6, 10, 9, 1, 3, 5, 2]

sorted_array = selection_sort(array)

print(sorted_array)

20. TUGAS Lakukan Pencarian untuk Solusi 5 Queen Problem dengan algoritma Backtracking!​


5 Queen Problem adalah masalah penempatan lima ratu di papan catur berukuran 5x5 sehingga tidak ada satu pun ratu yang dapat menyerang satu sama lain. Algoritma backtracking adalah salah satu cara untuk menyelesaikan masalah ini dengan menguji setiap kemungkinan penempatan ratu yang berbeda secara sistematis. Berikut adalah implementasi algoritma backtracking untuk 5 Queen Problem:

1. Inisialisasi papan catur 5x5 dengan seluruh nilainya = 0.

2. Mulai dengan baris pertama dan cari lokasi yang tersedia untuk memasukkan ratu pertama pada baris tersebut.

3. Jika tidak ada lokasi yang tersedia, kembali ke baris sebelumnya dan pindahkan ratu tersebut ke lokasi yang lain.

4. Jika terdapat lokasi yang tersedia, letakkan ratu pada sel tersebut dan tandai sel yang tidak dapat digunakan lagi oleh ratu lain.

5. Pindah ke baris berikutnya dan ulangi proses 2-4 untuk menempatkan ratu kedua, ketiga, keempat dan kelima.

6. Jika semua ratu berhasil ditempatkan, cetak solusinya.

Berikut adalah contoh implementasi algoritma backtracking dengan bahasa pemrograman Python:

```

board = [[0 for j in range(5)] for i in range(5)] # Inisialisasi papan catur 5x5

def is_valid_position(row, col):

# Cek apakah lokasi (row, col) bebas dari serangan ratu yang lain

for i in range(row):

if board[i][col] == 1:

return False

for i, j in zip(range(row, -1, -1), range(col, -1, -1)):

if board[i][j] == 1:

return False

for i, j in zip(range(row, -1, -1), range(col, 5, 1)):

if board[i][j] == 1:

return False

return True

def solve_queen_problem(row):

# Basis rekursi: semua ratu telah ditempatkan

if row == 5:

for i in range(5):

print(board[i])

return True

# Rekursi: coba tempatkan ratu pada baris saat ini

for col in range(5):

if is_valid_position(row, col):

board[row][col] = 1

solve_queen_problem(row+1)

board[row][col] = 0 # backtrack jika tidak ada solusi yang ditemukan

return False

solve_queen_problem(0) # Cari solusi dari baris pertama (0)

```

Output dari program di atas adalah semua solusi yang memenuhi kriteria, yaitu 10 kemungkinan penempatan ratu yang berbeda.

40. 1. 2. TUGAS INFORMATIKA KELAS VIII Buatlah algoritma teks dan algoritma pseudocode untuk membuat output seperti berikut dengan struktur pengulangan 1 4 9 16 25 36 49 64 81 100 Diketahui a=5 dan b-8. buatlah algoritma teks dan algoritma pseudocode sehingga a=8 dan b=5 dengan struktur dasar sekuensial (urutan) Ketentuan mengerjakan: 1. Kerjakan di buku tugas 2. Setelah selesai, buku tugas dikumpulkan di TU​


Penjelasan:

1. Mempermudah pengguna memahami cara penyelesaian suatu masalah

Fungsi utama dari pseudocode adalah untuk mempermudah manusia membaca atau menyelesaikan suatu permasalahan dari sebuah algoritma atau bahasa pemrograman. Hal ini bisa terjadi karena pseudocode dapat dibaca dengan mudah oleh penggunanya yaitu manusia.

Tanpa pseudocode, permasalahan akibat kesalahan penulisan bahasa pemrograman akan sangat sulit dideteksi. Namun hal ini bisa diatasi karena pseudocode bisa mengkonversi kode bahasa pemrograman. Sehingga ketika terjadi masalah atau error terhadap kode yang diterima atau sedang dibuat, pengguna dapat memanfaatkan pseudocode untuk mendeteksi masalah tersebut.

2. Alat dokumentasi penyelesaian masalah

Tidak hanya membantu menyelesaikan sebuah permasalahan, pseudocode juga bisa menjadi alat dokumentasi. Yaitu sebagai jembatan penghubung antara pengguna dan bahasa pemrograman. Pseudocode akan mendokumentasikan seluruh proses pengolahan data tersebut.

Pseudocode berperan penting dalam membantu pengguna menerjemahkan bahasa pemrograman dengan lebih cepat. Orang awam tidak bisa menganalisa tentang proyek pembuatan algoritma matematika atau suatu hal yang melibatkan bahasa pemrograman dengan komputer tanpa pseudocode.

Sehingga pseudocode menjadi alat dokumentasi yang membantu manusia menerjemahkan bahasa pemrograman ke dalam bahasa yang lebih sederhana agar dapat dimengerti oleh manusia.

3. Menuliskan rumus algoritma yang lebih mudah dibaca

Pseudocode juga berguna untuk menuliskan rumus-rumus algoritma dengan lebih sederhana. Sehingga algoritma yang ditulis dengan pseudocode menjadi lebih mudah dipahami daripada algoritma yang ditulis dengan bahasa pemrograman. Akhirnya, pengguna akan lebih cepat mendapat gambaran tentang tingkat kerumitan suatu program yang akan dibuat, dengan menuliskan algoritma tersebut dalam bentuk pseudocode.

4. Lebih ringkas dan praktis

Pseudocode lebih mudah dibuat karena tidak bergantung pada suatu sistem tertentu dan juga algoritmanya lebih ringkas dan mudah dibaca.
