2. Sebutkan salah satu teknologi dalam pengelolaan website pada level source code ! Berikan contoh dan paparkan secara jelas dalam alurnya


XSS – Cross Side Scripting

     Metode XSS ini pernah diulas di edisi ke-2 buletin Geek Factor kita ini. Secara prinsip, XSS menyisipkan sebuah tag-tag HTML khusus kedalam sebuah situs.

Tag-tag ini dapat dimasukkan kedalam server melalui mekanisme HTML FORM yang valid. Ini khususnya dapat terjadi ketika web developer dari situs tersebut lengah akan kemungkinan timbulnya serangan ini. Begitu tag tersebut berhasil disisipkan, maka ketika situs tersebut menampilkan data yang telah tercemar oleh script XSS tadi maka akan timbul beberapa resiko seperti:


o  User session yang dibajak. Ini memungkinkan attacker menyamar sebagai user tersebut.


o  Pencurian data-data user, khususnya login dan password.

 Bagi web developer, untuk mencegah serangan XSS sebetulnya mudah saja. Setiap input yang diterima melalui HTML FORM harus diperiksa apakah mengandung tag-tag yang dapat dianggap berbahaya. Tag itu lalu dibuang sebelum menyimpan datanya kedalam server. Penulis sarankan untuk membaca edisi ke-2 karena disana XSS diulas secara lebih mendetail.

7. apakah semua website berbasis client server? jelaskan dan beri contoh!​


Ya, seluruh website memiliki basis client server. Hal tersebut karena client server memiliki peran sebagai penghubung antara client yaitu pengguna, dengan server sebagai penyedia informasi yang diminta oleh pengguna. Contoh aplikasi yang menggunakan sistem client server adalah email.

Pembahasan

Client server adalah sebuah arsitektur jaringan yang menghubungkan sistem server dan sistem client yang berkomunikasi melalui jaringan komputer. Client adalah perangkat yang menerima dan akan menampilkan serta menjalankan aplikasi. Server adalah perangkat yang menyediakan serta bertindak sebagai pengelola aplikasi, data, dan keamanannya.

Prinsip kerja client server bergerak dua arah dan bertugas sebagai program web browser yang mengirimkan informasi kepada user yang ada di seluruh dunia. Untuk mengoperasikannya, client server membutuhkan IP address dari server dan halaman website.

22. website radio dan tv online menunjukan fungsi website sebagai sarana​


Jawaban:

Website radio dan TV online berfungsi sebagai sarana penyedia informasi, hiburan, dan interaksi antara penyiar dan pengguna. Beberapa fungsi penting dari website radio dan TV online antara lain:

1. Penyedia Informasi: Website radio dan TV online menyediakan informasi mengenai program acara, jadwal tayang, berita terkini, dan konten-konten yang relevan dengan layanan radio atau TV tersebut. Pengguna dapat dengan mudah mengakses informasi ini melalui website tersebut.

2. Menyediakan Siaran Langsung (Streaming): Melalui website, penyiar radio dan TV dapat menyediakan siaran langsung atau streaming acara secara online. Pengguna dapat menonton atau mendengarkan program acara yang sedang berlangsung melalui website ini.

3. Menyediakan Konten Beragam: Website radio dan TV online juga menyediakan berbagai konten berupa video, audio, artikel, dan podcast yang dapat diakses oleh pengguna kapan saja dan di mana saja. Pengguna dapat menikmati konten-konten ini sesuai dengan minat dan kebutuhan mereka.

4. Interaksi dengan Pengguna: Website radio dan TV online juga memberikan kesempatan bagi pengguna untuk berinteraksi dengan penyiar melalui fitur komentar, polling, dan kotak saran yang disediakan. Pengguna dapat memberikan tanggapan, saran, atau partisipasi dalam acara melalui website tersebut.

5. Kemudahan Akses dan Mobilitas: Website radio dan TV online memberikan kemudahan akses bagi pengguna untuk menonton atau mendengarkan program acara secara fleksibel melalui perangkat digital seperti laptop, smartphone, atau tablet. Pengguna dapat mengakses website ini di mana saja dan kapan saja selama terhubung dengan internet.

Dengan adanya website radio dan TV online, informasi dan hiburan dapat lebih mudah diakses oleh pengguna, serta memberikan pengalaman interaktif dan fleksibel dalam menikmati program acara yang disediakan.

29. 10 jenis file source code


10 jenis file source code.

Dalam pemrograman, source code adalah file dimana kode program ditulis. Setiap bahasa pemrograman dan IDE (Integrated Development Environment) memiliki ekstensi file sendiri. Misalnya:

1. *.cpp : jenis file source code berbahasa C++

2. *.c : jenis file source code berbahasa C

3. *.h : jenis file header source code berbahasa C, merupakan file yang berisi keterangan variabel dari source code c/c++

4. *.cls : jenis file source code dari class, biasanya ditulis menggunakan bahasa C#

5. *.java : jenis file source code bahasa java.

6. *.sln : jenis file source code dari solution yang di generate oleh program MS Visual Studio, berisi konfigurasi project.

7. *.html : jenis file source code halaman web

8. *.php : jenis file source code dari program backend website yang ditulis dengan bahasa php.

9. *.pas : jenis file source code berbahasa pascal

10. *.js : jenis file source code dari kode javascript
