6. Sebutkan dan jelaskan pengertian /24 dalam pembuatan alamat ip di mikrotik​


Jawaban:

/24 adalah angka yang merujuk pada subnetmask sebuah IP address. Subnetmask sendiri berfungsi untuk menentukan berapa jumlah host yang dapat terbentuk dengan memberikan sebuah network ID. /24 sendiri sama dengan 255.255.255.0. Katakanlah ada sebuah jaringan dengan network ID 192.168.1.0 dengan subnetmask /24. Dari network ID dan subnetmask tersebut, kita dapat menentukan jumlah host yang dapat terbentuk dan alamat broadcast suatu network. Jumlah host adalah 254 host dengan alamat broadcast adalah 192.168.1.255 dan jumlah address yang digunakan berada pada range 192.168.1.1 hingga 192.168.1.254.

Menentukkan jumlah host dapat dilakukan dengan mengurangi 256 dengan byte terakhir pada subnetmask kemudian dikurangi 2. Jadi 256-0=256-2 =254.

Alasan dikurangi dua karena address pertama dan terakhir digunakan sebagai network ID dan broadcast address yaitu 192.168.1.0 dan 192.168.1.255

37. Blok alamat IP yang digunakan oleh organisasi untuk menetapkan alamat host internal merupakan pengertian dari​


Blok alamat IP yang digunakan oleh organisasi untuk menetapkan alamat host internal biasanya disebut sebagai "Private IP Address" atau "Private Address Space". Private IP Address adalah rentang alamat IP yang ditetapkan secara eksklusif bagi penggunaan internal di jaringan komputer pribadi.

Ada tiga blok alamat IP yang diperuntukkan untuk penggunaan internal ini yaitu:

1. Blok alamat IP kelas A (10.0.0.0 - 10.255.255.255): Rentang alamat IP ini dapat digunakan oleh organisasi yang memiliki jaringan besar.

2. Blok alamat IP kelas B (172.16.0.0 - 172.31.255.255): Rentang ini diperuntukkan bagi organisasi yang memiliki jaringan berukuran menengah.

3. Blok alamat IP kelas C (192.168.0.0 - 192.168.255.255): Rentang ini sering digunakan oleh rumah tangga dan organisasi kecil yang memiliki jaringan kecil.

Alamat IP dalam blok ini hanya dapat digunakan secara internal di jaringan organisasi. Ketika paket data yang diberi alamat IP private ini akan dikirim melalui internet maka perlu dilakukan proses Network Address Translation (NAT) untuk menggantikan alamat IP private dengan alamat IP publik yang unik agar paket data bisa dikenali oleh router dan jaringan publik.

Dalam pengaturan jaringan blok alamat IP private ini memungkinkan organisasi untuk memiliki kontrol penuh terhadap jaringan internal mereka dan memberikan tambahan keamanan dengan tidak mempublikasikan alamat IP internal secara langsung ke internet.
