21. Sebuah negara kecil hanya terdiri dari 3 kelompok etnis. Kelompok etnis A memiliki 38478 orang, Komitmen kelompok etnis A ditambah kelompok etnis B memİIİkİ dua kali lebih banyak kelompok etnik C. Kelompok etnis C memiliki dua kali lebih banyak orang sebagai kelompok etnis B. Berapakah total populasi ketiga kelompok etnis tersebut?


Jawab:

76956

Penjelasan dengan langkah-langkah:

Terjemahkan maksud soal terlebih dahulu

A = 38478

Komitmen kelompok etnis A ditambah kelompok etnis B memİIİkİ dua kali lebih banyak kelompok etnik C

A + B = 2C

Kelompok etnis C memiliki dua kali lebih banyak orang sebagai kelompok etnis B

C = 2B

Substitusi

A + B = 2C

A + B = 2 (2B)

A + B = 4B

A = 3B

3B = 38478

B = 12826

C = 2B = 2 * 12826 = 25652

Maka Total

A + B + C = 38478 + 12826 + 25652

A + B + C = 76956

Jawaban:

Enam sifat-sifat suatu kelompok etnis :

Memiliki nama yang khas yang mengindentifikasikan hakikat dari suatu masyarakat. Suatu contoh misalnya suku Dayak, suku Batak,  suku Melayu dan sebagainya. Memiliki suatu mitos akan kesatuan nenek moyang. Mitos tersebut  biasanya terdapat ide dalam kesamaan asal-usul dalam waktu dan  tempat tertentu sehingga kelompok tersebut suatu kekeluargaan  yang fiktif. Kelompok tersebut mempunyai memori masa lalu yang sama  seperti para pahlawan, kejadian-kejadian tertentu di dalam hari-hari  peringatan suku-suku tersebut. Kelompok tesebut memiliki kesatuan elemen-elemen budaya  seperti agama, adat istiadat, bahasa. Kelompok tersebut terikat dengan suatu tanah tumpah darah (home land) baik secara fisik maupun hanya sebagai keterikatan simbolik  terhadap tanah leluhur seperti pada kelompok-kelompok diaspora. Memiliki suatu rasa solidaritas dari penduduknya ​
