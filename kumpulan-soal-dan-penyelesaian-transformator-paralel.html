6. buatlah soal transformator yang menghitung tegangan out put​


Jawaban:

1. Perbandingan lilitan primer dengan lilitan sekunder sebuah transformator adalah 4:10. Jika kuat arus primer 5 ampere, berapakah kuat arus sekunder?

Penyelesaian:

Diketahui:

NP : NS = 4 : 10,

IP= 5 A.

Ditanyakan: IS = ?

Jawab:

IS = (NP / NS) x IP

IS = (4/10) x 5

IS = 2 A

Jadi kuat arus sekundernya 1 Ampere.

2. Sebuah trafo digunakan untuk menaikkan tegangan AC dari 12 V menjadi 120 V. Hitunglah kuat arus primer, jika kuat arus sekunder 0,6 A dan hitunglah jumlah lilitan sekunder, jika jumlah lilitan primer 300.

Penyelesaian:

Diketahui:

Vp = 12 V

Is = 0,6 A

Vs = 120 V

Np = 300

Ditanya: IP = ... ? dan Ns= ... ?

Jawab:

Vp/Vs = Is/Ip

Ip = (Vs/Vp) x Is

Ip = (120 V/12 V) x 0,6 A

Ip = 6 A

Vp/Vs = Np/Ns

Ns = (Vs/Vp) x Ns

Ns = (120 V/12 V) x 300

Ns = 3000

Jadi, kuat arus primernya 0,6 A dan kumparan sekunder terdiri atas 3.000 lilitan.

3. Sebuah transformator dihubungkan dengan PLN pada tegangan 100 V menyebabkan kuat arus pada kumparan primer 10 A. Jika perbandingan jumlah lilitan primer dan sekunder 1 : 25, hitunglah tegangan pada kumparan sekunder dan kuat arus pada kumparan sekunder.

Penyelesaian:

Diketahui:

Vp = 100 V

Ip = 10 A

Np : Ns = 1 : 25

Ditanya: Vs = ... ? dan Is= ... ?

Jawab:

Vp/Vs = Np/Ns

Vs = (Ns/Np) x Vp

Vs = (25/1) x 100 V

Vs = 2.500 V

Np/Ns = Is/Ip

Is = (Np/Ns) x Ip

Is = (1/25) x 10 A

Is = 0,4 A

Jadi, tegangan sekundernya 2.500 V dan kuat arus sekundernya 0,4 A.

4. Sebuah trafo arus primer dan sekundernya masing-masing 0,8 A dan 0,5 A. Jika jumlah lilitan primer dan sekunder masing-masing 100 dan 800, berapakah efisiensi trafo?

Jawab:

Diketahui:

Ip = 0,8 A

Np = 1.000

Is = 0,5 A

Ns = 800

Ditanya: η = ... ?

Penyelesaian:

η = (Is x Ns/ Ip x Np) x 100%

η = (0,5 A x 800/ 0,8 A x 1000) x 100%

η = (400/ 800) x 100%

η = 0,5 x 100%

η = 50%

Jadi, efisiensi trafo sebesar 50%.

5. Sebuah trafo tegangan primer dan sekundernya 220 V dan 55 V. Jika kuat arus primer 0,5 A dan kuat arus sekunder 1,5, berapakah efisiensi trafo?

Jawab:

Diketahui:

Ip = 0,5 A

Vp = 220 V

Is = 1,5 A

Vs = 55 V

Ditanya: η = ... ?

Penyelesaian:

η = (Is x Vs/ Ip x Vp) x 100%

η = (1,5 A x 55 V/0,5 A x 220 V) x 100%

η = (82,5 W/ 110 W) x 100%

η = 0,75 x 100%

η = 75%

Jadi, efisiensi trafo sebesar 75%.

6. Efisiensi sebuah trafo 60%. Jika energi listrik yang dikeluarkan 300 J, berapakah energi listrik yang masuk trafo?

Jawab:

Diketahui:

η = 60%

Ws = 300 J

Ditanya: Wp = ... ?

Penyelesaian:

η = (Ws/Wp) x 100%

60% = (300 J/Wp) x 100%

60% = (300 J/Wp) x 100%

6 = 3000 J/Wp

Wp = 3000 J/6

Wp = 3000 J/6

Wp = 500 J

Jadi, energi yang masuk trafo sebesar 500 J.

7. Sebuah trafo memiliki efisiensi 75%. Tegangan inputnya 220 V dan tegangan outputnya 110 V. Jika kuat arus primer yang mengalir 2 A, berapakah kuat arus sekundernya?

Jawab:

Diketahui:

η = 75%

Vp = 220 V

Vs = 110 V

Ip = 2 A

Ditanya: Is = ... ?

Penyelesaian:

η = (Is x Vs/ Ip x Vp) x 100%

75% = (Is x 110 V/2 A x 220 V) x 100%

75 = 11.000 Is /440 A

Is = 75 x 440 A / 11.000

Is = 3 A

Jadi, kuat arus sekundernya sebesar 3 A

8. Sebuah trafo memiliki perbandingan lilitan 10 : 2 dihubungkan ke sumber listrik 100V untuk menyalakan sebuah lampu 25 W. Hitunglah tegangan listrik yang diserap oleh lampu dan kuat arus yang masuk kedalam trafo:

10 : 2 = 100 : Vs

Vs = 20 V

100 . Ip = 25

1. Perbandingan lilitan primer dengan lilitan sekunder sebuah transformator adalah 4:10. Jika kuat arus primer 5 ampere, berapakah kuat arus sekunder?

    Penyelesaian:

   Diketahui:

   NP : NS = 4 : 10,

   IP= 5 A.

   Ditanyakan: IS = ?

   Jawab:

   IS = (NP / NS) x IP

   IS = (4/10) x 5

   IS = 2 A

   Jadi kuat arus sekundernya 1 Ampere.


2.  Sebuah trafo digunakan untuk menaikkan tegangan AC dari 12 V menjadi 120 V. Hitunglah kuat arus primer, jika kuat arus sekunder 0,6 A dan hitunglah jumlah lilitan sekunder, jika jumlah lilitan primer 300.

    Penyelesaian:

    Diketahui:

    Vp = 12 V

    Is = 0,6 A

   Vs = 120 V

   Np = 300

   Ditanya: IP = ... ? dan Ns= ... ?

   Jawab:

   Vp/Vs = Is/Ip

   Ip = (Vs/Vp) x Is

   Ip = (120 V/12 V) x 0,6 A

   Ip = 6 A

       Vp/Vs = Np/Ns

   Ns = (Vs/Vp) x Ns

   Ns = (120 V/12 V) x 300

   Ns = 3000


   Jadi, kuat arus primernya 0,6 A dan kumparan sekunder terdiri atas 3.000 lilitan.


3.  Sebuah transformator dihubungkan dengan PLN pada tegangan 100 V menyebabkan kuat arus pada kumparan primer 10 A. Jika perbandingan jumlah lilitan primer dan sekunder 1 : 25, hitunglah tegangan pada kumparan sekunder dan  kuat arus pada kumparan sekunder.

    Penyelesaian:

   Diketahui:

   Vp = 100 V

   Ip = 10 A

   Np : Ns = 1 : 25

   Ditanya: Vs = ... ? dan Is= ... ?

   Jawab:

   Vp/Vs = Np/Ns

   Vs = (Ns/Np) x Vp

   Vs = (25/1) x 100 V

   Vs = 2.500 V

      

        Np/Ns = Is/Ip

    Is = (Np/Ns) x Ip

    Is = (1/25) x 10 A

    Is = 0,4 A

    Jadi, tegangan sekundernya 2.500 V dan kuat arus sekundernya 0,4 A.


4. Sebuah trafo arus primer dan sekundernya masing-masing 0,8 A dan 0,5 A. Jika jumlah lilitan primer dan sekunder masing-masing 100 dan 800, berapakah efisiensi trafo?

Suatu trafo step-up terpasang dengan sumber listrik dengan tegangan sebesar 200V dan daya sebesar 500kW. Perbandingan lilitan primer dan sekundernya adalah 1 : 10. Jika daya yang hilang adalah sebesar 15%, maka tentukan arus dan tegangan pada kawat sekunder!

Solusi :

Diketahui :

[tex]V_P=200\,V\\P_P=500\,kW=500000W\\N_P : N_S = 1 : 10\\\eta=100\%-15\%=85\%[/tex]

Ditanya :
[tex]V_S=?\,V\\I_S=?\,V[/tex]

Jawab :

Mencari tegangan sekunder :

[tex]\frac{N_P}{N_S}=\frac{V_P}{V_S}\\\\\frac{1}{10}=\frac{200\,V}{V_S}\\\\V_S=200\,V\times 10\\\\V_S=2000\,V[/tex]

Jadi, tegangan sekundernya adalah sebesar 600V

Mencari arus sekunder :

Karena efisiensinya 85%, maka daya sekundernya

[tex]\eta=\frac{P_S}{P_P}\times100\%\\\\85\%=\frac{P_S}{500000\,W}\times100\%\\\\P_S=\frac{85\%\times500000\,W}{100\%}\\\\P_S=425000\,W[/tex]

Maka, arus sekundernya adalah

[tex]P_S=V_S\,I_S\\\\425000\,W=600\,V\times I_S\\\\I_S=\frac{425000\,W}{2000\,V}\\\\I_S=212,5\,A[/tex]

Jadi, arus sekundernya adalah sebesar 212,5 ampere

32. diketahui sebuah transformator mempunyai efisiensi 85% jika sumber listrik berdaya 900 Watt dilewatkan pada transformator tersebut. berapa daya maksimum yang keluar dari transformator.mohon di jawab besok di kumpul​


Daya maksimum yang keluar dari transformator tersebut adalah 765 Watt.

Pembahasan :

Efisiensi adalah daya kerja dari transformator untuk menaikkan suatu tegangan. Apabila efisiensi < 100% merupakan transformator step-down, karena daya keluaran lebih kecil dari daya masuk. Apabila efisiensi = 100% merupakan transformator ideal, karena daya masuk sama dengan daya keluar. Apabila efisiensi > 100% merupakan transformator step-up, karena daya keluaran lebih besar dari daya masuk. Secara umum, dapat dirumuskan dengan :

[tex]\boxed {\bold {\eta = \frac {P_{out}}{P_{in}} \times 100\%}} \: [/tex]

Dengan ketentuan :

η = efisiensi transformator (%) P (in) = daya masuk (Watt) P (out) = daya keluaran (Watt) Langkah Penyelesaian :

Diketahui :

η = efisiensi transformator = 85 % P (in) = daya masuk = 900 Watt

Ditanya : P (out) = daya keluaran =... Watt

Jawaban :

Hilangkan sementara tanda % untuk mempermudah penghitungan.

[tex]\eta = \frac {P_{out}}{P_{in}} \times 100\% \\ 85\cancel \% = \frac {P_{out}}{900} \times 100\cancel \% \\ 85 \times 900 =P_{out} \times 100 \\ \frac{85 \times 9\cancel{00}}{1\cancel{00}} = P_{out} \\ 765 \: watt = P_{out} \: [/tex]

Jadi, daya maksimum yang keluar dari transformator tersebut adalah 765 Watt.

34. Sebuah transformator memiliki jumlah lilitan primer Dan sekunder adalah 1000 lilitan Dan 1200 lilitan jika kumpulan primer transformator diberi tegangan 200 volt mama tegangan yang dihasilkan transformator adalah


Jumlah lilitan primer dan sekunder  1.000  :  1.200  dan Tegangan primer sebesar 200 Volt berapa tegangan yang dihasilkan Transformator ....



Penjelasan Dengan langkah-langkah :

Transformator adalah alat utnuk menaikkan atau menurunkan tegangan,Alat ini bekerja dengan menggunakan induksi elektromagnetik.  

Bagian -bagian Transformator terdiri atas kumparan primer , kumparan sekunder , dan inti besi lunak.  

Kumparan primer bagian trafo yang terhubung dengan sumber tegangan. inti besi lunak berfungsi untuk tempat melingkarnya kumparan primer dan sekunder. Tegangan dan arus keluaran trafo mengalir dari kumparan sekunder.  

Transformator ada 2 macam yaitu  

1. Transformator step up  

Transformatoryang berfungsi menaikkan tegangan.  

Ciri-ciri transformator step up  

1. NP < Ns = Lilitan primer lebih sedikit dari lilitan sekunder  

2. VP < Vs = Tegangan primer lebih kecil dari tegangan sekunder  

3. IP > IS = Arus Primer lebih besar dari arus sekunder  

2. Transformator step down  

Ciri-ciri transformator step down  

1. NP > NS = Lilitan Primer Lebih besar dari lilitan sekunder.  

2. VP > VS = Tegangan primer lebih besar dari tegangan sekunder  

3. IP < IS = Arus primer lebih kecil dari arus sekunder  

Dari hubungan jumlah lilitan, tegangan, dan arus secara matematis di Rumuskan sebagai berikut  

[tex]\frac{VP}{VS}[/tex] = [tex]\frac{NP}{NS}[/tex] = [tex]\frac{IS}{IP}[/tex]

Untuk efisiensi Transformator di Rumuskan sebagai berikut :  

μ = [tex]\frac{PS}{PP}[/tex] x 100%  

μ = [tex]\frac{VS.IS}{VP.IP}[/tex] x 100%  

μ = [tex]\frac{NS.IS}{NP.IP}[/tex] x 100%  

Keterangan  

VP = Tegangan Primer (volt)

VS = Tegangan Sekunder (volt)

NP = Lilitan Primer  

NS = Lilitan Sekunder

IS = Arus Sekunder ( A )

IP = Arus Primer (A)

PS = Daya Sekunder (watt)

PP = Daya Primer (watt)

μ = Efisiensi (%)  

==================================================================Pembahasan soal  

Diketahui :  

NP = Lilitan primer = 1.000

NS = Lilitan Sekunder = 1.200

VP = Tegangan Primer = 200 Volt


Ditanyakan :

VS = Tegangan Sekunder = ...


Penyelesaian :

Gunakan Rumus :

       NP/NS =  VP/VS

1000/1.200 = 200/VS

             VS = 1.200/1000 x 200

             VS =  240

Jadi Tegangan Sekunder sebesar 240 Volt
