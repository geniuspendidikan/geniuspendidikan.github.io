18. Sebutkan dan jelaskan 3 keamanan dari windows 2008!


Kelebihan Windows Server 2008 adalah: 
1. Windows Server 2008 dapat beroperasi tanpa tampilan grafis atau graphical user interface(GUI) dengan adanya teknologi powershell. 
2. Pengguna dapat memilih fungsi-fungsi yang dibutuhkannya saja atau menambah fungsi lainnya jika membutuhkan sewaktu-waktu tanpa melakukan instalasi ulang. 
3. Kemampuan virtualisasi bahkan embedded (menyatu) dengan Windows Server 2008. 
4. Windows Server 2008 mampu mengatur besar bandwidth yang dapat dipakai setiap aplikasi maupun komputer yag terhubung ke jaringan. 
5. Windows Server 2008 juga sanggup mengontrol keamanan jaringan dengan fitur Network Access Protection. 
6. Server juga dapat mengatur setiap akses identitas ke jaringan agar aman dan praktis dengan adanya fitur read only domain controller. 
7. Melalui powershell, administrator tetap dapat memantau komputer di jaringan dari jarak jauh. 
8. Lebih aman dalam mengendalikan laju informasi. 
9. Peningkatan Kapasitas Server untuk melayani lebih Simultan Koneksinnya. 
10. Driver disk yang fault toleran yang mendukung disk mirroring dan disk stripping dengan parity (RAID 1 dan RAID 5). 
11. Bebas dari Kode 16 Bit milik MS-Dos,mendukung operasi 32 bit dan semua Fitur yang ditawarkan oleh Microprosesor 32 bit seperti dapat mengamati memori hingga 4 Gb dan Terproteksi. 
12. Di Desain agar kompatibel dengan Sistem Operasi terdahulu seperti MS-Dos,IBM OS/2. 
13. Peningkatan kemampuan layanan server TCP/IP seperti DHCP,WNS dan DNS. 
14. Tool untuk mengintegrasikan Netware dan memonitoring Jaringan. 
15. Model keamanan berbasis Domain penuh. 
16. Terdapat Layanan untuk Macintosh. 
17. Bisa Membooting jarak jauh untuk client. 
18. Terintegrasi Paket Back Office. 
19. Terdapat Network Client Administrator. 
20. Fitur pengendalian yang lebih baik (more control). Yaitu fitur yang dapat membuat perusahaan memegang kontrol yang lebih terhadap server mereka.

38. cari pengertian windows dan android​


— Windows adalah sebuah sistem operasi yang dikembangkan oleh perusahaan Microsoft. Ini adalah salah satu sistem operasi paling umum digunakan di dunia, terutama pada komputer pribadi (PC) dan laptop. Windows memiliki berbagai versi, seperti Windows 7, Windows 10, dan Windows 11 (hingga tanggal pemotongan pengetahuan saya pada September 2021). Sistem operasi Windows dirancang untuk berbagai keperluan, termasuk penggunaan rumahan, bisnis, dan pengembangan perangkat lunak.

— Android adalah sistem operasi yang dikembangkan oleh Google, dan banyak digunakan pada perangkat seluler seperti smartphone dan tablet. Android juga digunakan pada perangkat lainnya seperti televisi pintar, konsol permainan, dan banyak perangkat berbasis layar sentuh. Android bersifat open source, yang berarti kode sumbernya dapat diakses oleh pengembang, memungkinkan mereka untuk membuat berbagai aplikasi yang berjalan pada perangkat yang menjalankan sistem operasi ini. Android juga memiliki berbagai versi, dengan nama-nama seperti Jelly Bean, KitKat, Lollipop, dan seterusnya.


39. Kegunaan PPTPD sever


1.Koneksi PPTP VPN

PPTP menjadi pilihan populer jaringan virtual private network karena kemudahan dalam setting maupun penggunaan. Kamu tidak perlu melakukan instalasi VPN client agar bisa terhubung dengan jaringan server PPTP VPN. Koneksi Point to Point Tunneling Protocol bisa berjalan di berbagai sistem operasi seperti Microsoft Windows, Linux, MacOS X, iPhone OS, dan Android OS sudah memiliki built in protokol ini.

Kamu hanya membutuhkan nama server, vpn account meliputi nama pengguna dan kata sandi untuk dapat terhubung dengan server PPTP VPN. Koneksi jenis ini paling banyak digunakan pada area wifi hotspot untuk melindungi privasi online dan mencegah pihak lain yang ingin meretas perangkat smartphone atau tablet yang kamu gunakan di area hotspot.
