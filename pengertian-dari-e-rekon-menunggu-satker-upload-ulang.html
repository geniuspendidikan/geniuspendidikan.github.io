Pengertian Upload

Upload dapat juga disebut dengan Unggah, Upload merupakan kebalikan dari Download. Pengertian Upload adalah proses mengirim file yang tersimpan dari komputer lokal ke komputer sistem jaringan Internet. Secara umum upload dapat diartikan sebagai proses transmisi data dari pemakai komputer/komputer client ke jaringan internet.

Atau upload adalah mengunggah file atau data (bisa berupa dokumen, foto, audio, video) dari komputer pribadi ke sebuah sistem server, dan file atau data tersebut akan dipublikasikan di internet, sehingga data yang telah diupload tersebut dapat dilihat, di download (diambil) oleh orang lain.

Kembali ke Menu Pembahasan ↑

Contoh Upload

Contoh Upload yang biasa kita lakukan pada facebook kita sering memasang /mempublikasikan foto, nah proses memasang foto itu namanya upload/ mengunggah, file yang di upload itu bisa di download dan disimpan oleh pengguna komputer lain yang sedang terkoneksi Internet dengan catatan file yang di upload tidak diproteksi.

Kembali ke Menu Pembahasan ↑

Tujuan Upload

Tujuan dari upload adalah supaya pengguna internet yang berada di dunia bisa melihat, memanfaatkan atau menggunakan file yang kita upload. Hal tersebut dapat kita contohkan orang lain yang dapat menikmati karya tulisan dalam bentuk data atau file yang kita pajang pada blog, media sosial atau media lainnya. Selain itu tujuan upload yaitu untuk menyimpan data pada sebuah server agar apabila kita kehilangan data tersebut pada komputer kita, kita bisa mencarinya kembali pada server tersebut.

Kembali ke Menu Pembahasan ↑

Manfaat Upload

Manfaat upload file yaitu kita bisa menghemat memori yang diperlukan untuk menyimpan sebuah file karena file kita pindahkan dengan cara upload ke dalam sebuah server yang menyediakan penyimpanan data atau file. Selain itu file yang telah kita upload sewaktu-waktu bisa kita ambil lagi dengan cara mendownloadnya. Manfaat lainnya yaitu kita tidak perlu khawatir jika file yang kita simpan pada komputer atau flash disk atau penyimpanan lain terhapus/ rusak karena file yang kita upload tersebut akan lebih aman dan bisa kita dowload sewaktu-waktu.

Kembali ke Menu Pembahasan ↑

Cara Upload File di Internet

Apabila kita ingin mengupload, terlebih dahulu kita hubungkan perangkat komputer atau handphone ke internet, kemudian pilih penyimpanan server yang kamu sukai, seperti google drive, drop box, 4shared, onedrive, mediafire, media sosial dan masih banyak yang lainnya. Setelah memilih server yang kamu sukai, kamu harus membuat akun dengan menggunakan email kamu. Setelah kamu mempunyai akun, kamu dapat mengupload file ke server tersebut.

Diagram Batang

Diagram batang sesuai dengan namanya, diagram tersebut berbentuk batang. Tata letaknya yaitu dengan menempatkan nama objek di samping kiri, dan frekuensi dituliskan di bagian bawah. Jadi kita lambang kan seperti huruf "L". Dimana; l untuk nama objek nya, dan _ untuk banyak data/Frekuensi. Tinggal sesuaikan objeknya dengan berapa banyak frekuensi, lalu gambarlah batangnya sesuai batas frekuensinya.

Diagram Garis

Hampir sama seperti diagram batang, hanya saja bukan menggunakan batang yg berbentuk persegi panjang melainkan menggunakan garis lurus.

Diagram Lingkaran

Diagram Lingkaran bisa menerapkan 2 sistem, yaitu derajat dan persen. Cara mengubah data murni ke bentuk desimal dan persen yaitu:

30. Jelaskan pengertian upload dan dwonload


Jawaban:

Download adalah suatu proses transmisi sebuah file atau data dari sebuah sistem komputer ke sistem komputer yang lainnya. Dari internet, user/pengguna yang melakukan proses download (unduh) yaitu proses dimana seorang user meminta atau menrequest sebuah file dari sebuah komputer lain (seperti: web site,server atau yang sebagainya) dan lalu menerimanya.

Upload adalah sering dikatakan juga dengan Unggah, yaitu merupakan suatu cara untuk mengirimkan File (contohnya seperti file: gambar, program, musik, dokumen pdf, dan lain-lain) dari komputer pribadi ke sebuah sistem Server dan file atau data akan di publikasikan di internet sehingga data yang telah di upload dapat dilihat dan diambil (download) oleh orang lain atau orang banyak.

Atau definisi Upload yang lainnya adalah suatu proses transmisi sebuah file atau Data dari sebuah sistem komputer pribadi ke sistem komputer yang lainnya (server) dengan arah yang berkebalikan dengan download. Dari jaringan internet, user yang melakukan proses upload yaitu proses dimana user mengirimkan file ke komputer lain yang meminta file. Singkatnya proses transmisi atau pengiriman file ataupun data, dari komputer pribadi ke jaringan internet.

32. Struktur teks cerita ulang(rekon) terbagi tiga, tulislah satu penggalan cerita ulang pribadimu dalam bagian insiden cerita!


Jawaban:

Teks cerita ulang disebut juga dengan teks biografi, karena teks tersebut menceritakan kembali pengalaman seorang tokoh yang dikisahkan tokoh tersebut kepada orang lain dan orang lain menuliskan kisah tersebut dalam sebuah teks.

Penjelasan:

Struktur teks cerita ulang atau teks biografi adalah:

1. Orientasi atau pendahuluan, yang berisi tentang data umum tokoh utama yang diceritakan pada teks.

2. Peristiwa penting, yang berisi tentang urutan kejadian-kejadian atau pengalaman-pengalaman yang dialami seorang tokoh.

3. Reorientasi. Pada teks cerita ulang bersifat opsional. Penutup dapat berisi pendapat penutup tentang tokoh yang ditulisnya. Atau penutup dapat berupa prestasi-prestasi yang diperoleh oleh tokoh secara urut.
