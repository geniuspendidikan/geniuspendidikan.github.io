5. Rumus to be going to


Kalimat positif :
Subyek + to be + going to + verb1
contoh :
I’m going to go swimming tomorrow.

Kalimat negatif
Subyek + to be + not + going to + verb1
Contoh :
He is not going to play football.

Kalimat interrogatif
To be + subyek + going to + verb1?
Contoh :
Are you going to Banyuwangi next week?

9. Contoh kalimat tenses be going to present (+)(-)(?) dan be going to past (+)(-)(?)...


*Present
pattern:
(+) = S + V1(s/es) + O(Adj./Adv/etc.)
(-) = S + Do/Does + not + V1 + O
(?) = Do/Does + S + V1 + O ?
Example :
(+) = He buys a car now
(-) = He does not buy a car now
(?) = Does he buy a car now ?

Note: keterangan waktu tidak harus ditulis.

*Past
pattern:
(+) = S + V2 + O
(-) = S + Did + not + V1 + O
(?) = Did + S + V1 + O ?
Example:
(+) = She went to school yesterday
(-) = She did not go to school yesterday
(?) = Did she go to school yesterday ?

Note: keterangan waktu tidak harus ditulis

16. jelaskan tentang pengguna "to be Going to"​


Penggunaan Will dan TO BE GOING TO

Contoh:

1. According to the weather report, it will be cloudy tomorrow.

2. According to the weather report, it is going to be cloudy tomorrow.

(Artinya: Berdasarkan laporan cuaca, hari besok akan berawan)

CATATAN:

Ketika si pembicara sedang melakukan prediksi (suatu pernyataan tentang apa yang dia pikirkan akan benar atau akan terjadi kedepannya), baik will atau to be going to sama-sama bisa digunakan.

Contoh (1) dan (2) mempunyai makna yang sama.

3. Be carefull! You will hurt yourself!

4. Watch out! You are going to hurt yourself!

Contoh (3) and (4) mempunyai makna yang sama.

Fungsi TO BE GOING TO

TO BE GOING TO digunakan untuk mengekspresikan Prior Plan (rencana terdahulu).

Contoh:

A: Why did you buy this paint?

B: I am going to paint my bedroom tomorrow.

Answer:

B. Subject + to be + going to + verb 1

Future simple tense :

(+) S + to be (am/is/are) + going to + V1

(-) S + to be (am not/is not (isn't)/are not (aren't)) + going to + V1

(?) To be (am/is/are) + S + going to + V1?

I = am

He/she/it = is

You/they/we = are
